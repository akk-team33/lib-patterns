Java Development by Andreas Kluge-Kaindl, Bremen (de)

de.team33.patterns

This project consists of several library modules, each dealing with narrowly defined problems and their solutions. The implementation is based on the idea that each individual solution can be used without unnecessary or excessive dependencies. In fact, there are no dependencies outside of this project - except for pure testing purposes and of course a Java Runtime Environment.

In the current version, the modules require a JRE of version 8 or higher.
